# 📄 HOW TO CONVERT TEXT TO WORD DOCUMENT FOR SUBMISSION

## QUICK CONVERSION STEPS

### Option 1: Using Microsoft Word (Recommended)

**Step 1: Open Word**
- Click Start → Microsoft Word → New Blank Document

**Step 2: Copy the Content**
- Navigate to: `C:\xampp\htdocs\FairFareSystem\PROJECT_PROPOSAL_ACADEMIC.txt`
- Open the file with Notepad (right-click → Open with → Notepad)
- Select all content: `Ctrl + A`
- Copy: `Ctrl + C`

**Step 3: Paste into Word**
- In Word, paste: `Ctrl + V`
- The document will populate automatically

**Step 4: Set Margins**
- Go to: File → Page Setup → Margins
- Set as follows:
  - Top: 1.57 inches (40mm)
  - Left: 1.57 inches (40mm)  
  - Right: 0.98 inches (25mm)
  - Bottom: 0.98 inches (25mm)

**Step 5: Format Text**
- Select all: `Ctrl + A`
- Font: Times New Roman
- Size: 12pt
- Line Spacing: 1.5
- Alignment: Justified

**Step 6: Format Headings**
- Find "CHAPTER" headings (Ctrl+F)
- Select each heading
- Size: 16pt
- Bold: `Ctrl + B`
- Make UPPERCASE

**Step 7: Add Page Numbers**
- Insert → Page Number → Keep position at bottom/center
- On first page, uncheck "Show number on first page"
- Start numbering from page 2 (where Chapter One begins)

**Step 8: Create Table of Contents**
- Place cursor on new page after Abstract
- References → Table of Contents → Choose style
- Word auto-generates from heading styles

**Step 9: Save**
- File → Save As
- Name: `FAIRFARE_SYSTEM_PROPOSAL_[Your Name].docx`
- Location: Desktop or project folder
- Format: Word Document (.docx)

## IF DOCUMENT FORMATTING LOOKS OFF

**Problem: Text is jumbled or spacing is wrong**
- Solution: Delete problematic content, manually copy-paste section by section
         Use Paste Special (Ctrl+Shift+V) → Unformatted Text

**Problem: Margins keep reverting**
- Solution: In Word, go to File → Page Setup
         Under "Default" make sure margins apply to "Whole Document"

**Problem: Page numbers appearing on all pages**
- Solution: Double-click in footer area
         Select "First Page Different"
         Delete page number from first page

**Problem: Word/Doxc refuses to open text file**
- Solution: Open Notepad first
         Then File → Open the text file
         Copy-paste into Word
