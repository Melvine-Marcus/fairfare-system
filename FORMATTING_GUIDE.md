# How to Format the Academic Proposal for Final Submission

## Quick Guide: Converting TXT to Properly Formatted Document

### Step 1: Open the Text File in Microsoft Word

1. Open Microsoft Word
2. Go to **File → Open**
3. Select `PROJECT_PROPOSAL_ACADEMIC.txt` from the FairFareSystem folder
4. Word will ask about encoding - select **UTF-8** encoding
5. Click **Open**
6. The document will insert with default formatting

### Step 2: Set Up Page Margins (CRITICAL)

1. Click **Layout** tab (in ribbon)
2. Click **Margins** → **Custom Margins**
3. Set the following in the dialog:
   - **Top:** 40 mm (or 1.57 inches)
   - **Left:** 40 mm (or 1.57 inches)
   - **Right:** 25 mm (or 0.98 inches)
   - **Bottom:** 25 mm (or 0.98 inches)
4. Click **OK**

### Step 3: Set Font and Line Spacing for Entire Document

1. Select all text: **Ctrl+A**
2. In the **Home** ribbon:
   - Font: Change to **Times New Roman**
   - Font Size: Change to **12 pt**
   - Line spacing: Click line spacing icon → **1.5**
3. Alignment: Click **Justify** button

### Step 4: Format Cover Page

1. Position cursor at the beginning (before "FAIRFARE SYSTEM")
2. Select the cover page content (first 10-12 lines)
3. In **Home** ribbon:
   - Font Size: Change to **16 pt**
   - Text: Convert to UPPERCASE (select text → **Format** → **Text** → **Change Case** → **UPPERCASE**)
   - Alignment: **Center**

### Step 5: Page Numbers and Pagination

1. Go to **Insert** tab
2. Click **Page Numbers** → **Bottom of Page** → **Center alignment**
3. The dialog: "Link to Previous" must be unchecked
4. Double-click on page 1 footer and delete the number
5. Double-click on page 1 footer and add: "Declarations start here"
6. Double-click on page 2 in the footers - now page 2 will show "2" (first numbered page)

**Alternative Method:**
1. Insert page numbers on all pages
2. Go to page 1 footer, right-click on number
3. Select **Field** → **Options** → Offset: **-1**
4. This will start counting from header page as page 1 (displays as empty)

### Step 6: Add Page Breaks Between Major Sections

Position cursor before each chapter heading and press **Ctrl+Enter** to insert page break:
- Before "COVER PAGE"
- Before "DECLARATION"
- Before "DEDICATION"
- Before "ACKNOWLEDGEMENT"
- Before "TABLE OF CONTENTS"
- Before each new chapter

### Step 7: Format Chapter Headings

For each chapter heading (CHAPTER ONE, CHAPTER TWO, etc.):
1. Select the heading
2. Format as:
   - Font Size: **14 pt**
   - Font Weight: **Bold**
   - Alignment: **Left** or **Center** (your choice)
   - Spacing After: **12 pt**

### Step 8: Format Subheadings

For all subheadings (Introduction, Background, etc.):
1. Select subheading
2. Format as:
   - Font Size: **12 pt**
   - Font Weight: **Bold**
   - Spacing Before: **12 pt**
   - Spacing After: **6 pt**

### Step 9: Format Tables and Code Blocks

Tables in the document:
1. Select table
2. Right-click → **Table Properties**
3. Set font to **11 pt**
4. Align to center
5. Add table caption: **Table X: [Description]**

Code blocks/diagrams:
1. Select code/diagram section
2. Font: **Courier New** or **Consolas**
3. Font Size: **10 pt**
4. Line Spacing: **Single** (1.0)
5. Background: Light gray (optional, right-click → **Borders and Shading**)

### Step 10: Add Table of Contents

1. Position cursor where you want TOC (after "TABLE OF CONTENTS" heading)
2. Go to **References** tab → **Table of Contents** → Choose a style
3. Word will automatically populate chapters and headings
4. If your headings don't appear, ensure they're formatted as "Heading 1" or "Heading 2"

To format existing text as headings for TOC:
1. Select a chapter title
2. On **Home** tab, select "Heading 1" from the styles list
3. Repeat for all chapter titles

### Step 11: Add Page Breaks for Print

Before printing:
1. Add page break before **CHAPTER ONE** if not already present
2. Add page break before **CHAPTER TWO**, etc.
3. Ensure proper spacing between sections

### Step 12: Review and Adjust

1. **View → Print Layout** to see how it will print
2. Check:
   - All margins are correct
   - Page numbers start on page 2
   - Headings are properly formatted
   - Tables are readable
   - Page breaks are in right places
   - No orphaned text at bottom of pages

### Step 13: Add Screenshots Before Final Submission

Add these to **Chapter Three - System Design**:
1. Screenshots of system interfaces:
   - Login page
   - User dashboard
   - Incident reporting form
   - Admin dashboard
   - View incidents table

To insert screenshots:
1. Go to **Insert** → **Pictures** → **This Device**
2. Select screenshot file
3. Resize as needed (maintain aspect ratio)
4. Add caption: Select image → **References** tab → **Insert Caption**
5. Set caption format: "Figure X: [Description]"
6. Position caption below image

### Step 14: Add References (APA Format)

At the end of document, add a section titled **REFERENCES**

APA Format Examples:

**Book:**
Author, A. A. (Year). Title of work. Publisher.

**Journal Article:**
Author, A. A., & Author, B. B. (Year). Title of article. Journal Title, Volume(Issue), page range.

**Website:**
Author, A. A. (Year). Title of page. Retrieved from http://www.example.com

**Software/System:**
Developer Name. (Year). Title of software (Version #) [Software]. Publisher.
https://doi.org/xxxxx (if available)

### Step 15: Save in Multiple Formats

1. **Save as PDF** (for submission):
   File → **Save As** → Format: **PDF** → Save
   
   This ensures:
   - Formatting is preserved
   - Fonts embed correctly
   - Pages look identical on all devices

2. **Save as DOCX** (for editing):
   File → **Save As** → Format: **Word Document (.docx)** → Save

### Step 16: Final Quality Check

Before submission, print or preview as PDF:

✅ Margins: Top/Left 40mm, Right/Bottom 25mm
✅ Font: Times New Roman, 12pt, black text
✅ Spacing: 1.5 lines throughout
✅ Page numbers: Start from page 2
✅ Titles: 16pt on cover, 14pt for chapters
✅ Tables: Properly formatted and numbered
✅ Page breaks: Before each major section
✅ Alignment: Body text justified
✅ No spelling/grammar errors
✅ All figures have captions

## Troubleshooting Common Issues

### Issue: Text not displaying in Courier New
**Solution:** Install Courier New font or use Consolas as alternative

### Issue: Headers/footers different on each page
**Solution:** Check "Different First Page" is unchecked in Design tab

### Issue: Table of Contents blank
**Solution:** Ensure chapter headings are formatted as Heading 1 or Heading 2 style

### Issue: Page numbers starting from page 1 of content
**Solution:** Either format first page as different from the rest, or manually delete page number from first page footer

### Issue: Text overlapping due to margins
**Solution:** Check if you used inches (1.57") vs millimeters (40mm) - they're different!

### Issue: PDF looks different from Word
**Solution:** Save as PDF from Word using **File → Export as PDF** to ensure preservation
